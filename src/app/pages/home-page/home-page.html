SFMS - Malware Scanning Service

SFMS (Secure File Management System) offers a Class-2 standard global malware scanning service hosted in Europe, designed to provide an on-demand malware scanning solution accessible via both the internet and intranet. Leveraging a flexible REST API, SFMS seamlessly integrates with front-end and back-end systems, ensuring broad platform compatibility. Operating within a secure, isolated environment in the Allianz Global Network (AGN), this innovative service offers robust protection against zero-day and advanced malware threats. SFMS not only detects and prevents security breaches but also repairs infected files, delivering a comprehensive layer of security to users worldwide.

The diagram represents the high level architecture of the SFMS , which is hosted within an isolated infrastructure in the Allianz Global Network (AGN). This strategic setup ensures that data remains securely within the organization's network, preventing external exposure and enhancing data privacy. The service is accessible from both the internet and intranet, providing flexibility and broad accessibility to users. The SFMS API employs the standard OAuth 2.0 specification, a widely recognized protocol for secure authentication and authorization. By implementing OAuth 2.0, SFMS ensures that only authorized users can access the service, maintaining the integrity and security of data transactions.

Key Benefits :

Enhanced Security Layer

SFMS acts as a robust barrier, effectively screening and sanitizing potentially harmful files prior to their integration into the system, thus fortifying the overall security posture.

Cost Efficiency

Shared licensing costs translate to significant savings, offering an economical solution for file management needs.

Vendor Independence

SFMS liberates users from vendor dependencies, providing autonomy and flexibility in choosing solutions.

Zero-Knowledge System

SFMS operates on a zero-knowledge principle, meaning it does not retain or have access to sensitive business data. This ensures maximum confidentiality and privacy, instilling trust and confidence in users regarding the security of their information.

Reusability Across Applications

SFMS provides reusable solutions that can be seamlessly integrated into diverse applications, offering consistency and efficiency across the organizational ecosystem. This versatility ensures compatibility with various systems and enhances operational agility

Extensible Design

SFMS features an extendable design architecture, allowing for scalability and customization to accommodate evolving business needs and technological advancements. This adaptability future-proofs the system, ensuring long-term viability and relevance.
